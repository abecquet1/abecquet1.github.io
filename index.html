Recherche des images

Les liens vers les diférents chapitres sont maintenant décorés avec des images, pour l'instant issues de librairies d'images libres de droit. Je propose à celles et ceux qui le veulent de m'envoyer leurs créations personnelles afin de les remplacer. Vous pourrez déposer vos proposisions dans un repo github que je vais créer pour l'occasion. Les consignes sont les suivantes :
  • format d'image : 300 x 300 px ;
  • avoir un lien avec le chapitre illustré ;
  • s'intégrer dans les couleurs choisies pour le site.
