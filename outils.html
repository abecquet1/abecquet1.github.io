Outils

Les outils à installer pour travailler chez soi

Installer Python

Le langage de programmation étudié en NSI est le Python. Rendez-vous ici pour en télécharger la dernière version.

Pour écrire des programmes, nous avons besoin d'un environnement de dévelopement (IDE). Il s'agît d'un éditeur de fichier spécialisé dans l'écriture de code. Python en propose un par défaut : IDLE. Si l'installation de Python s'est bien déroulée, IDLE devrait être installé sur votre ordianteur.

D'autres IDE généralistes ou spécialisés existent (Visual Studio Code, Pycharm, etc.). À vous de les essayer pour trouver celui qui vous convient le mieux.

Se créer un compte Github

Afin de pouvoir travailler à la fois en classe et à la maison, individuellement ou en groupe, nous allons utiliser le service Github. À la manière d'un Google drive, Github permet de disposer d'un espace de stockage en ligne et propose de plus des outils spécialisés dans le développement de logiciels (travail collaboratif, versionnage, etc.). Pour s'inscrire, rendez-vous ici.

Sur votre oridnateur personnel, vous pouvez installer Github Desktop qui vous permet d'intéragir avec votre compte Github sans passer par un navigateur internet.
